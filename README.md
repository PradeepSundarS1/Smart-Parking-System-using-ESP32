# Smart Parking System with Fire Detection using ESP32

## Project Title
- Smart Parking System with Fire Detection using ESP32

---

## Abstract
- The Smart Parking System with Fire Detection is an IoT-based mini project developed using ESP32 to improve parking management and safety monitoring.
- The system automatically controls the entrance gate using an IR sensor and a servo motor.
- When a vehicle is detected at the entrance, the gate opens automatically and closes after a few seconds.
- The project also monitors parking slot occupancy in real-time using IR sensors.
- For safety purposes, a flame sensor and buzzer are integrated to detect fire and provide instant alerts.
- A modern web dashboard is developed using the ESP32 web server for real-time monitoring and control.

---

## Working Principle
- IR sensor detects vehicle presence at the entrance gate.
- ESP32 processes the sensor input and controls the servo motor.
- The gate opens automatically when a vehicle is detected and closes after a few seconds.
- Four IR sensors continuously monitor parking slot occupancy.
- Slot information is updated in real-time on the web dashboard.
- Flame sensor detects fire or flame presence.
- Buzzer activates immediately during fire detection.
- ESP32 web server hosts a dashboard for live monitoring.
